سياسة الخصوصية

تاريخ السريان: 19 فبراير 2026
تطبيق رصيدي فورجي هو أداة مساعدة تتيح للمستخدم عرض معلومات حسابه لدى مزود الخدمة بطريقة أسهل عبر جهازه الشخصي. التطبيق غير تابع أو معتمد أو ممثل رسمياً لأي شركة اتصالات أو مزود خدمة ولا يمثل أي جهة حكومية.

البيانات التي يتم إدخالها

يقوم المستخدم بإدخال رقم الخدمة (رقم الهاتف) داخل التطبيق بهدف الاستعلام عن معلومات الرصيد أو بيانات الحساب المرتبطة به. لا يقوم التطبيق بإنشاء حسابات مستخدمين ولا يطلب الاسم أو البريد الإلكتروني أو أي بيانات تعريف شخصية.

كيفية استخدام البيانات

  • يتم استخدام رقم الخدمة لإرسال طلب استعلام إلى واجهة الويب الخاصة بمزود الخدمة بنفس الآلية التي يتم بها الاستعلام عبر الموقع الرسمي باستخدام المتصفح.
  • يقوم التطبيق بمعالجة الاستجابة المستلمة وعرض المعلومات للمستخدم داخل التطبيق فقط.
  • لا يتم تخزين البيانات على خوادم المطور، ولا يتم نقل البيانات إلى أي طرف ثالث، وجميع عمليات المعالجة تتم محلياً داخل جهاز المستخدم.

التخزين المحلي

قد يقوم التطبيق — بشكل اختياري — بحفظ بعض البيانات داخل جهاز المستخدم فقط مثل: سجل الاستعلامات، والأرقام المفضلة. هذه البيانات تبقى داخل جهاز المستخدم ويمكن حذفها من داخل التطبيق في أي وقت أو عند إزالة التطبيق.

الأذونات

يستخدم التطبيق الأذونات التالية فقط:

  • الإنترنت (INTERNET): لاستخدام الاتصال بالشبكة من أجل إرسال طلبات الاستعلام وعرض المعلومات للمستخدم.

لا يستخدم التطبيق أي أذونات للوصول إلى جهات الاتصال أو الرسائل أو الموقع أو الملفات الشخصية أو أي بيانات حساسة أخرى.

مشاركة وحذف البيانات

لا يقوم التطبيق بجمع أو تخزين أو مشاركة أي بيانات مستخدم على خوادم المطور. جميع عمليات الاستعلام تتم بين جهاز المستخدم وواجهة الخدمة الخاصة بمزود الخدمة عبر الإنترنت.

بما أن التطبيق لا يخزن بيانات المستخدمين على خوادم خارجية، فإن حذف التطبيق من جهاز المستخدم يؤدي إلى حذف جميع البيانات المخزنة محلياً بشكل نهائي.

الأطفال

التطبيق غير موجه للأطفال دون سن 13 عاماً ولا يجمع أي معلومات منهم بشكل مقصود.

الأمان والتعديلات

لا يحتفظ المطور بأي قواعد بيانات للمستخدمين أو معلومات حسابات خارج أجهزة المستخدمين.

قد يتم تحديث سياسة الخصوصية عند الحاجة. سيتم نشر أي تحديث في نفس الصفحة مع تعديل تاريخ السريان.
